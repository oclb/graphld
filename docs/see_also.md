# See Also

## Related Projects

### LDGM Repository

Main LDGM repository with MATLAB API:

- [https://github.com/awohns/ldgm](https://github.com/awohns/ldgm)

### graphREML (MATLAB)

Original graphREML implementation in MATLAB:

- [https://github.com/huilisabrina/graphREML](https://github.com/huilisabrina/graphREML)

We recommend using the Python implementation in this repository, which is much faster.

### LD Score Regression

Reference implementation of LD score regression:

- [https://github.com/bulik/ldsc](https://github.com/bulik/ldsc)

### LDGM-VCF (bcftools)

Giulio Genovese's bcftools plugin with LDGM-VCF file format specification and C implementation:

- [https://github.com/freeseek/score](https://github.com/freeseek/score)

### SuiteSparse

Sparse matrix operations used by graphld:

- [https://github.com/DrTimothyAldenDavis/SuiteSparse](https://github.com/DrTimothyAldenDavis/SuiteSparse)
