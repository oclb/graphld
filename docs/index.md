# GraphLD

GraphLD provides LDGM-backed tools for working with GWAS summary statistics.

You can use it from the command line for graphREML, BLUP, clumping, simulation, and surrogate-marker workflows, or from Python for lower-level LDGM and likelihood operations.

## Features

- **graphREML** for heritability partitioning and annotation enrichment from GWAS summary statistics.
- **Enrichment score tests** for fast post-hoc annotation testing from precomputed derivative statistics.
- **LDGM matrix operations** for precision-matrix and Schur-complement workflows without forming dense LD matrices.
- **Simulation** for generating GWAS summary statistics under flexible mixture and annotation-dependent architectures.
- **LD clumping** for thresholding and pruning variants into approximately independent index sets.
- **BLUP weights** for infinitesimal-model effect-size estimation.
- **Surrogate markers** for handling variants that are absent from the LDGM reference panel.
